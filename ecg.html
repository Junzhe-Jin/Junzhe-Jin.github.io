ECG Anomaly Detection

Data Deviation Value Judgment Model Based on ECG Big Data.

Background & Motivation

Electrocardiogram (ECG) is a crucial diagnostic tool for identifying heart diseases by recording the electrical signals generated by the heart. However, the sheer volume of continuous ECG data generated in clinical settings makes manual analysis highly time-consuming and prone to human error. To address this challenge, it is essential to establish an accurate and automated comprehensive analysis model capable of handling the massive volume of ECG signals to determine the patient's cardiac status computationally.

Algorithm Design & Implementation

The pipeline begins with a rigorous signal denoising process: low-frequency baseline drift is removed using polynomial detrending, and high-frequency noise is suppressed via the soft thresholding of smooth wavelet transforms. This significantly improves signal clarity for downstream morphological analysis.

Subsequently, the Dynamic Time Warping (DTW) algorithm is applied to align successive cardiac cycles, mitigating the effects of temporal variations in heartbeats. A slope-threshold method is then utilized to extract critical morphological features from the aligned signals. Finally, to identify irregular heartbeats, the system employs a hierarchical clustering-based model combined with Local Outlier Factor (LOF) scoring.

Research Value & Impact

This unsupervised learning framework successfully isolates abnormal ECG patterns from normal physiological variations without relying on extensively labeled medical datasets. By effectively combining signal processing techniques (Wavelet Transform, DTW) with machine learning anomaly detection algorithms (LOF), this research provides a robust and scalable methodological foundation for automated healthcare monitoring and early-stage arrhythmia diagnosis in big data streams.
